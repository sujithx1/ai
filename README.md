# 🧠 AI Academy – Learn AI by Playing

A gamified, interactive learning platform designed for developers transitioning into AI engineering. Designed like a blend of **Duolingo, Notion, and Discord**, this platform teaches Artificial Intelligence concepts through play, interactive code sandboxes, mini projects, and quizzes.

---

## 🚀 Key Features

*   🎮 **Interactive Playgrounds:** Train models dynamically, simulate AI winters, and play with classic algorithms in real-time.
*   🏆 **Gamified System:** Earn Experience Points (XP) and unlock badges as you complete quizzes and challenges.
*   🎌 **Dev-friendly Analogies:** High-level ideas explained using anime (Naruto, Dragon Ball, One Piece) and pop-culture references.
*   💻 **TypeScript & React Sandbox:** Step-by-step code tutorials highlighting rule-based logic vs machine learning models.
*   🌌 **Premium Dark UI:** Modern theme featuring HSL colors, glassmorphic card panels, and smooth Framer Motion transitions.

---

## 📚 Completed Lessons

1.  **🚀 What is AI anyway?**
    *   Concepts: Tokenizers, probability brain, rules vs weights.
    *   Playground: Dataset size vs accuracy prediction simulator.
    *   Mini Project: Semantic spam/phishing checking filter.
2.  **⏳ The History of AI**
    *   Concepts: Turing Test, Dartmouth workshop, the two AI Winters, Deep Learning GPU boom, and Transformers.
    *   Playground: Interactive AI Winter funding & hype simulator.
    *   Mini Project: Retro pattern-matching ELIZA chatbot (1966 style).
3.  **🤖 Types of AI**
    *   Concepts: ANI (Narrow), AGI (General), ASI (Super) categories alongside Reactive, Limited Memory, and Self-Aware classifications.
    *   Playground: App categorizer widget (Siri, HAL 9000, Skynet).
    *   Mini Project: AI Type Identifier using custom feature matrix configs.
4.  **🤖 Machine Learning (ML)**
    *   Concepts: Supervised (labels), Unsupervised (clusters), and Reinforcement (rewards/penalties) branches.
    *   Playground: Accuracy vs data size optimizer training bar.
    *   Mini Project: Collaborative movie suggestion filter.

## 🛠️ Tech Stack

*   **Runtime/Package Manager:** [Bun](https://bun.sh/) / Node.js
*   **Framework:** [React 18](https://react.dev/)
*   **Build Tool:** [Vite](https://vitejs.dev/)
*   **Language:** [TypeScript](https://www.typescriptlang.org/)
*   **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
*   **Animations:** [Framer Motion](https://www.framer.com/motion/)
*   **Icons:** [Lucide Icons](https://lucide.dev/)

## 📦 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Local Dev Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```
